# Government Job Database Analysis (SQL EDA Project)

# Overview
This project performs Exploratory Data Analysis (EDA) on a government job dataset using SQL. The goal is to extract meaningful insights about job vacancies, candidates, salaries, and hiring trends.

# Tools & Technologies
SQL (MySQL)
Excel / CSV Dataset
GitHub for version control

# Dataset
The dataset includes:
Job details (title, vacancies, salary)
Candidate information
Exam results
application details

# Key Analysis Performed
Total vacancies by job role
Top candidates based on performance
Average salary analysis
Pass/Fail statistics
Job competition insights

# High-demand job roles
 Sample Insights
"Editor, Film/Video" has the highest vacancies
 IT roles show strong demand
 Some exams have very high pass rates (easy exams)

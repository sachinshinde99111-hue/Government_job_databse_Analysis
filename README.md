# Government_job_databse_Analysis

# Project Overview
This project focuses on analyzing a Government Job Database to extract meaningful insights about job postings, candidates, applications, salaries, vacancies, and selection trends. The goal is to help understand hiring patterns and improve decision-making using data analysis techniques.

# Objectives
Analyze government job postings and vacancies
Study candidate applications and selection trends
Identify high-demand job roles
Generate business insights from job data

 # Technologies Used
MYSQL
Python 
NumPy
SQL (for database queries)
Jupyter Notebook

# Dataset Description
The dataset contains multiple tables such as:
Jobs Table – job titles, vacancies, salary
Candidates Table – candidate details
Applications Table – job applications
Exams Table – exam dates and exam marks
result table - exam results and status

# SQL Queries Used

This project includes advanced SQL queries like:

SELECT, WHERE, GROUP BY, ORDER BY
JOIN operations
Subqueries (IN, EXISTS)
Aggregate functions (COUNT, AVG, MAX, MIN)
